# TalentScope – AI Resume Analyzer (Java)

## 📌 Project Overview

**TalentScope – AI Resume Analyzer** is a simple Java-based tool that analyzes resume text and evaluates the candidate's technical skills.
The system scans the resume for important programming and technology keywords, calculates a **resume score**, and provides **suggestions for improvement**.

This project demonstrates basic concepts of **Java programming, string processing, arrays, and collections**.

---

## 🚀 Features

* Resume text analysis
* Automatic **resume scoring system**
* Detection of **technical skills**
* Identification of **missing important skills**
* Suggestions to improve resume quality
* Simple and beginner-friendly Java implementation

---

## 🛠 Technologies Used

* **Java**
* Java Collections Framework
* String Processing
* Scanner class for user input

## ⚙️ How It Works

1. The user pastes resume text into the program.
2. The program scans for predefined technical skills such as:

   * Java
   * Python
   * SQL
   * HTML
   * CSS
   * JavaScript
   * React
   * Machine Learning
   * Data Structures
   * Algorithms
   * Git
   * Linux
3. Each detected skill increases the **resume score**.
4. The system displays:

   * Resume score
   * Skills found
   * Missing skills
   * Suggestions for improvement

## 💻 How to Run the Project

### Step 1

Clone the repository

```
git clone https://github.com/yourusername/talentscope-resume-analyzer.git
```

### Step 2

Navigate to the project folder

```
cd talentscope-resume-analyzer
```

### Step 3

Compile the Java program

```
javac ResumeAnalyzer.java
```

### Step 4

Run the program

```
java ResumeAnalyzer
```
